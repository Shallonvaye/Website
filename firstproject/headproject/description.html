HTML

HTML Elements
They are like code it often appear in pairs and has a particular function
HTML tag
Are elements surrounded by angle brackets
angle bracket
Are signs such as less than and greater than
HTML attribute
They are used to add more information to an HTML element
HTML comment
They contain information for the webmaster to see,but not for the viewer of the web page
HTML object element
They insert objects instead of enclosing text
HTML special characters
They are marks and other symbols
